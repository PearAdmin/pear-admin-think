* composer update
* 定义应用运行目录public
* 访问install.php进行安装


#### 代码一键生成CURD方法
>env APP_DEBUG = ture

* 第一步.创建多级
* 第二步.根据多级创建对应前缀数据表。
* 第三步.选择多级数据表生成。
* 建议定义软删除delete_time，自动生成回收站功能。如不需要可自行删除。
